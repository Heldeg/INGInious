Understand your result.

Here you will know what means the result you receive when you submit code.

After you submit the code, you will get an overall result, plus a grade an the result corresponding to each test case.

  • ACCEPTED

    You can breath with this result. This means that your output is the same as the expected output.

  • WRONG ANSWER

    This result means that the output given by your code does not coincide with the expected output.
    I some cases you can see the difference of your output and expected output. So that way, you can determine what is wrong with your code.

  • COMPILATION ERROR

    This message means that your code does not compile, that means there is something wrong with code. In order to help you with that, there you have some help telling you what is what is wrong with your code and where you can find the error.

  • RUNTIME ERROR

  • SUBMISSION TIMED OUT

  • MEMORY LIMIT EXCEEDED

  • INTERNAL ERROR
